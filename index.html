PyTorch Beginner 5：通过示例学习PyTorch

Posted on 2018-09-14 | In PyTorch

当经过前面PyTorch闪电战的入门洗礼，你已经具备了最基本PyTorch知识。而本章将继续介绍PyTorch的3大概念Tensors，Autograd和nn module，并以实例展示出来。教程原文请参考Learning PyTorch with Examples

Tensors

从numpy说起

在了解PyTorch之前，相信很多人就已经对numpy不陌生了。Numpy是科学计算的通用框架，它具有应用的广泛性，但在深度学习中，它并未提供任何针对性设计。不过，我们仍然可以通过手动设计，实现网络的前向和后向传递，完成一个简单的深度网络。
‘’’python

-- coding: utf-8 --

import numpy as np

N表示batch size；D_in为输入维数；H为隐藏层维数；D_out为输出维数。

N, D_in, H, D_out = 64, 1000, 100, 10

随机制造fake数据

x = np.random.randn(N, D_in)
y = np.random.randn(N, D_out)

随机初始化权值

w1 = np.random.randn(D_in, H)
w2 = np.random.randn(H, D_out)

手动设置学习率

learning_rate = 1e-6

训练迭代

for t in range(500):

# 前向传播
h = x.dot(w1)
h_relu = np.maximum(h, 0) # np.maxmun相当于ReLU函数
y_pred = h_relu.dot(w2)

# 计算损失
loss = np.square(y_pred - y).sum()
print(t, loss)

# 反向传播为w1，w2计算梯度
grad_y_pred = 2.0 * (y_pred - y)
grad_w2 = h_relu.T.dot(grad_y_pred)
grad_h_relu = grad_y_pred.dot(w2.T)
grad_h = grad_h_relu.copy()
grad_h[h < 0] = 0
grad_w1 = x.T.dot(grad_h)

# 更新权值
w1 -= learning_rate * grad_w1
w2 -= learning_rate * grad_w2

‘’’

PyTorch: Tensors

Numpy是一个很棒的框架，但它不能利用GPU来加速其数值计算。对于现代深度神经网络，GPU通常提供50倍或更高的加速，所以不幸的是，numpy对于现代深度学习来说还不够。
PyTorch Tensor在概念上与numpy数组相同，它是一个n维数组，PyTorch提供了许多用于在这些Tensors上运算的函数。在幕后，Tensors可以跟踪计算图和梯度，但它们也可用作科学计算的通用工具。与numpy不同，Tensors可以利用GPU加速其数值计算。要在GPU上运行Tensor，只需将其转换为新的数据类型即可。
我们先向numpy那样手动设计网络前向和反向传播。
‘’’python

-- coding: utf-8 --

import torch

dtype = torch.float
device = torch.device(“cpu”) # 用CPU训练

device = torch.device(“cuda:0”) # 或者也可以用GPU

N, D_in, H, D_out = 64, 1000, 100, 10

x = torch.randn(N, D_in, device=device, dtype=dtype)
y = torch.randn(N, D_out, device=device, dtype=dtype)

w1 = torch.randn(D_in, H, device=device, dtype=dtype)
w2 = torch.randn(H, D_out, device=device, dtype=dtype)

learning_rate = 1e-6
for t in range(500):

# 前向传播
h = x.mm(w1)
h_relu = h.clamp(min=0)
y_pred = h_relu.mm(w2)

# 计算损失
loss = (y_pred - y).pow(2).sum().item()
print(t, loss)

# 反向传播为w1，w2计算梯度
grad_y_pred = 2.0 * (y_pred - y)
grad_w2 = h_relu.t().mm(grad_y_pred)
grad_h_relu = grad_y_pred.mm(w2.t())
grad_h = grad_h_relu.clone()
grad_h[h < 0] = 0
grad_w1 = x.t().mm(grad_h)

# 更新权值
w1 -= learning_rate * grad_w1
w2 -= learning_rate * grad_w2

‘’’

Autograd

Tensors和autograd

在前面例子中，我们手动设计了网络前向和反向传播，对于一个简单的网络来说，这样做并没有任何困难，但在面对一个十分复杂的网络设计时，手动设计网络前向和反向传播变得不现实。
我们希望可以系统自动计算神经网络中的反向传播，而PyTorch中的autograd包便提供了这样的功能。使用autograd时，网络的正向传递将定义计算图，图中的节点是张量，边是从输入张量产生输出张量的函数。通过此图反向传播，便可自动计算梯度。
‘’’python

-- coding: utf-8 --

import torch

dtype = torch.float
device = torch.device(“cpu”) # 使用CPU训练

device = torch.device(“cuda:0”) # 或者也可以用GPU

N, D_in, H, D_out = 64, 1000, 100, 10

x = torch.randn(N, D_in, device=device, dtype=dtype)
y = torch.randn(N, D_out, device=device, dtype=dtype)

w1 = torch.randn(D_in, H, device=device, dtype=dtype, requires_grad=True)
w2 = torch.randn(H, D_out, device=device, dtype=dtype, requires_grad=True)

learning_rate = 1e-6
for t in range(500):

# 前向传播
y_pred = x.mm(w1).clamp(min=0).mm(w2)
# 计算损失
loss = (y_pred - y).pow(2).sum()
print(t, loss.item())
# 反向传播，计算梯度
loss.backward()
# 梯度更新
with torch.no_grad():
    w1 -= learning_rate * w1.grad
    w2 -= learning_rate * w2.grad
    # 梯度清零
    w1.grad.zero_()
    w2.grad.zero_()

‘’’

定义新的autograd函数

在这个例子中，自定义autograd函数来实现ReLU非线性，并使用它来设计网络。
‘’’python

-- coding: utf-8 --

import torch

class MyReLU(torch.autograd.Function):
@staticmethod
def forward(ctx, input):
ctx.save_for_backward(input)
return input.clamp(min=0)

@staticmethod
def backward(ctx, grad_output):
    input, = ctx.saved_tensors
    grad_input = grad_output.clone()
    grad_input[input < 0] = 0
    return grad_input

dtype = torch.float
device = torch.device(“cpu”)

device = torch.device(“cuda:0”)

N, D_in, H, D_out = 64, 1000, 100, 10

x = torch.randn(N, D_in, device=device, dtype=dtype)
y = torch.randn(N, D_out, device=device, dtype=dtype)

w1 = torch.randn(D_in, H, device=device, dtype=dtype, requires_grad=True)
w2 = torch.randn(H, D_out, device=device, dtype=dtype, requires_grad=True)

learning_rate = 1e-6
for t in range(500):
relu = MyReLU.apply # 调用自定义的ReLU函数

# 前向传播
y_pred = relu(x.mm(w1)).mm(w2)

# 计算损失
loss = (y_pred - y).pow(2).sum()
print(t, loss.item())

# 反向传播，计算梯度
loss.backward()

# 更新权值
with torch.no_grad():
    w1 -= learning_rate * w1.grad
    w2 -= learning_rate * w2.grad
    # 梯度清零
    w1.grad.zero_()
    w2.grad.zero_()

‘’’

训练分类器

PyTorch对于视觉处理有一个专门的包，包括一些如MNIST等常见数据集的数据加载器，以及对于图像的数据转换器，torchvision.datasets和torch.utils.data.DataLoader。
官网以CIFAR10数据集为例，演示了多种物体的图像识别。CIFAR-10中的图像尺寸为3×32×32，即尺寸为32×32像素的3通道彩色图像。教程中按顺序执行以下步骤：
1.使用torchvision加载和标准化CIFAR10训练和测试数据集
2.定义卷积神经网络
3.定义损失函数
4.在训练数据上训练网络
5.在测试数据上测试网络

加载和标准化CIFAR10

‘’’python
import torch
import torchvision
import torchvision.transforms as transforms

transform = transforms.Compose( # 组合多个变换
[transforms.ToTensor(), # 转换为张量
transforms.Normalize((0.5,0.5,0.5),(0.5,0.5,0.5))]) # 正态分布的均值与方差

trainset = torchvision.datasets.CIFAR10(root=’./data’,train=True, # 训练集
download=True,transform=transform) # 数据转换方式
trainloder = torch.utils.data.DataLoader(trainset, batch_size=4,
shuffle=True, num_workers=2) #数据加载的子进程数

testset = torchvision.datasets.CIFAR10(root=’./data’, train=False, # 测试集
download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4,
shuffle=False, num_workers=2)

classes = (‘plane’, ‘car’, ‘bird’, ‘cat’,
‘deer’, ‘dog’, ‘frog’, ‘horse’, ‘ship’, ‘truck’)
‘’’

定义卷积神经网络

‘’’python
import torch.nn as nn
import torch.nn.functional as F

class Net(nn.Module):
def init(self):
super(Net,self).init()
self.conv1 = nn.Conv2d(3,6,5) # in_ch, out_ch, kernel_size=55
self.pool = nn.MaxPool2d(2,2) # kernel_size=22, stride=2
self.conv2 = nn.Conv2d(6,16,5)
self.fc1 = nn.Linear(1655,120) # in, out
self.fc2 = nn.Linear(120,84)
self.fc3 = nn.Linear(84,10) # out put class=10
def forward(self,x):
x = self.pool(F.relu(self.conv1(x)))
x = self.pool(F.relu(self.conv2(x)))
x = F.relu(self.fc1(x))
x = F.relu(self.fc2(x))
x = self.fc3(x)
return x

net = Net()
‘’’

定义损失函数

‘’’python
import torch.optim as optim

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(),lr=0.001,momentum=0.9)
‘’’
Momentum的方法是在参数更新中加入动量项，对那些当前梯度与上次梯度相同的参数进行加强，反之削弱。这种方法可以获得更快收敛和较少振荡。

训练网络

‘’’python
for epoch in range(2):

running_loss = 0.0
for i,data in enumerate(trainloader,0):
    inputs, labels = data
    optimizer.zero_grad() # 梯度清零

    outputs = net(input) # 前向传播
    loss = criterion(outputs,labels) # 计算损失
    loss.backward() # 反向传播
    optimizer.step() # 参数更新

    running_loss +=loss.item()
    if i % 2000 == 1999: # 每2000个mini-batch 打印一次
        print('[%d,%5d] loss: %.3f' %(epoch + 1, i + 1, running_loss /2000))
        running_loss = 0.0

print(‘Finished Training’)
‘’’

测试网络

‘’’python
correct = 0
total = 0
with torch.no_grad():
for data in testloader:
images, labels = data
outputs = net(images)
_, predicted = torch.max(outputs.data, 1) # 返回行最大值和最大值对应的位置
total += labels.size(0)
correct += (predicted == labels).sum().item()

print(‘Accuracy of the network on the 10000 test images: %d %%’ % (
100 * correct / total))
